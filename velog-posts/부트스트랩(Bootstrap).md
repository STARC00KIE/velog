<h1 id="부트스트랩bootstrap">부트스트랩(Bootstrap)</h1>
<ul>
<li>통계학과 머신러닝에서 사용되는 샘플링 기법</li>
<li>원래 표본에서 복원추출(한 번 뽑은 대상을 추출 대상에서 포함)을 여러 번 반복하여 다양한 분석을 수행하는 방식</li>
<li>데이터가 적거나 정규분포를 따르지 않는 경우에도 신뢰할 수 있는 통계적 추론이가능함</li>
<li>계산이 간단함</li>
<li>표본분포가 비대칭일 때는 부정확할 수 있음</li>
</ul>

<h1 id="퍼센타일백분위-수">퍼센타일(백분위 수)</h1>
<ul>
<li>데이터가 작은 값에서 큰 값을 정렬되어 있을 때, p 번째 퍼센타일은 전체 데이터 중 아래에서 p%에 해당하는 위치의 값</li>
</ul>

<h1 id="부트스트랩-종류">부트스트랩 종류</h1>
<h2 id="기본-부트스트랩">기본 부트스트랩</h2>
<ul>
<li>원본 표본으로부터 동일한 크기의 재표본을 수천 번 추출(샘플링)하여 통계량을 계산하는 방법</li>
<li>그 통계랑들의 분포로부터 신뢰구간을 계산<h2 id="퍼센타일-부트스트랩">퍼센타일 부트스트랩</h2>
</li>
<li>부트스트랩 분포 자체의 분위수를 그대로 신뢰구간으로 삼는 방법</li>
<li>부트스트랩으로 얻은 추정량을 오름차순으로 정렬한 후, 분위수 값을 사용하여 결과 도출</li>
<li>직관적이고, 분포가 비정규적이여도 적용 가능</li>
<li>왜도가 심할 때는 보정이 필요함<h2 id="bca-부트스트랩">BCA 부트스트랩</h2>
</li>
<li>퍼센타일 부트스트랩 개선</li>
<li>편향 보정과 가속도를 적용한 방법, 정교하고 많이 쓰임</li>
<li>표본 분포의 왜도 조정하여 고려함</li>
<li>가장 정확하고 일반적으로 추천되는 신뢰구간 방법</li>
<li>특히 표본 크기가 작거나 분포가 비대칭적일 때 효과적</li>
</ul>
